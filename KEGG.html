Connecting KEGG pathways and STRING

1. Get the organism identifier

http://rest.kegg.jp/list/organism

Example:

T01001	hsa	Homo sapiens (human)	Eukaryotes;Animals;Vertebrates;Mammals

2. Get all pathways in the organism

http://rest.kegg.jp/list/pathway/[organism_id]

Example: http://rest.kegg.jp/list/pathway/hsa

path:hsa00010	Glycolysis / Gluconeogenesis - Homo sapiens (human)

3. Query each pathway

http://rest.kegg.jp/get/[pathway_id]
(KGML differs from the flat file ?)

  • Get the list of genes in the pathway

Example: http://rest.kegg.jp/get/path:hsa00010

GENE        3101  HK3; hexokinase 3 [KO:K00844] [EC:2.7.1.1]
            3098  HK1; hexokinase 1 [KO:K00844] [EC:2.7.1.1]
            3099  HK2; hexokinase 2 [KO:K00844] [EC:2.7.1.1]
...

4. Use STRING API to map the gene names to the external IDs (also available in STRING)

https://string-db.org/api/[output-format]/get_string_ids?identifiers=[identifiers]&[optional]

  • output-format = tsv
  • optional:
    • species (NCBI taxon identifier)
    • limit = 1 (return the best match)

Example:
(organism name <-> NCBI taxon ID mapping should be somehow automated) https://www.ncbi.nlm.nih.gov/Taxonomy/TaxIdentifier/tax_identifier.cgi

  • Homo sapiens NCBI taxon ID is 9606

https://string-db.org/api/tsv/get_string_ids?identifiers=HK3%0dHK1%0dHK2&species=9606&limit=1

queryIndex	stringId	ncbiTaxonId	taxonName	preferredName	annotation
